# MachineLearning

Jag har kört följande installationer i terminalen:
pip install pandas: För att läsa in CSV-filer, läsa tabellern etc.
pip install scikit-learn: huvudbiblioteketet för machinelearning i python, används för att träna modeller och utvärdera resultat etc.
pip install matplotlib: Hitta samband i data, visualisera mönster och visa resultat etc.



## Installations Guide (VS Code + Windows)

### 1. Klona projektet
1. Öppna **VS Code**
2. Klicka på **Clone Git Repository**
3. Klistra in repo-URL
4. Välj en plats på datorn
5. Klicka **Open** när VS Code frågar

> Projektet laddas nu ner och sparas lokalt på din dator.
